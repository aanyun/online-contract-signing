PROJECT TRACKER SAMPLE

  • A.  Project Description: The upper left side of this box contains all of the basic project information. It shows the project title, a brief project description, the production start date, total man hours, and progress bars for both the overall progress as well as the progress for each department.
  • B.  Contact Info: This block holds all the information about the contacts for this project. If you need to change any of your information, you can click the Edit icon and make any updates.
  • C.  Completion Statistics:  In this block, you will find the dates related to the completion of your eContent. The Review Period refers to how many days you have collectively to review your eContent, the Train Anyone Period refers to how many complimentary days you will have to train your clients, and the Estimated Completion Date is the projected date for when Ignitor Labs should have your eContent completed.
  • D.  Sort by Type:  You can sort the tasks a few ways. The History option allows you to look at all tasks, past and present, and see their status. The Current Tasks shows you all the tasks that are currently active. The Completed Tasks will show you only the completed tasks.
  • E.  Downloads & Printing  In the history view, you can download or print the project task table as well as download the revision template to your computer. Download Revision Template will save a copy of the revision template to your computer, Download Tracker Log will create an excel file that can be saved to your computer and Print Tracker Log will make a print-ready version of the table for you.
  • F.  Project Task Table  This table lets you see all of the tasks in a project, the last statuses given by Ignitor Labs and the Client, start and end times, and elapsed times.
